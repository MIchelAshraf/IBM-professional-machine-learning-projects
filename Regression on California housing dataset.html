The dataset for this project was derived from the 1990 U.S. census, using one row per census block group. A block group is the smallest geographical unit for which the U.S. Census Bureau publishes sample data (a block group typically has a population of 600 to 3,000 people). The data I investigate here consists of records about the houses in california/

Main objective of the analysis is to focus on prediction. In this project, I will be using different regression algorithms to find relationship between the house price and its features . best candidate algorithm with the best results will be chosen. The goal with this implementation is to construct a model that accurately predicts the price of each house given its features
